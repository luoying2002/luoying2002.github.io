rm жҳҜ Linux е’Ңзұ» Unix ж“ҚдҪңзі»з»ҹдёӯзҡ„дёҖдёӘеҹәжң¬е‘Ҫд»ӨпјҢз”ЁдәҺеҲ йҷӨж–Үд»¶жҲ–зӣ®еҪ•гҖӮе®ғжҳҜ вҖңremoveвҖқ зҡ„зј©еҶҷгҖӮrm е‘Ҫд»ӨеҸҜд»ҘеҲ йҷӨеҚ•дёӘж–Үд»¶гҖҒеӨҡдёӘж–Үд»¶жҲ–ж•ҙдёӘзӣ®еҪ•пјҢдҪҶдҪҝз”Ёж—¶йңҖиҰҒйқһеёёе°ҸеҝғпјҢеӣ дёәеҲ йҷӨж“ҚдҪңйҖҡеёёжҳҜдёҚеҸҜйҖҶзҡ„гҖӮ

д»ҘдёӢжҳҜеҜ№ rm е‘Ҫд»Өзҡ„иҜҰз»Ҷд»Ӣз»ҚпјҢеҢ…жӢ¬еҹәжң¬з”Ёжі•гҖҒеёёз”ЁйҖүйЎ№е’ҢзӨәдҫӢгҖӮ

еҹәжң¬иҜӯжі•

  • ж–Үд»¶/зӣ®еҪ•пјҡжҢҮе®ҡиҰҒеҲ йҷӨзҡ„ж–Үд»¶жҲ–зӣ®еҪ•и·Ҝеҫ„гҖӮ
  • йҖүйЎ№пјҡз”ЁдәҺжҺ§еҲ¶еҲ йҷӨиЎҢдёәпјҢдҫӢеҰӮйҖ’еҪ’еҲ йҷӨгҖҒејәеҲ¶еҲ йҷӨгҖҒдәӨдә’зЎ®и®ӨзӯүгҖӮ

еёёз”ЁйҖүйЎ№

йҖүйЎ№ еҠҹиғҪ
-i еҲ йҷӨеүҚйҖҗдёҖиҜўй—®зЎ®и®ӨгҖӮ
-f ејәеҲ¶еҲ йҷӨпјҢдёҚжҸҗзӨәпјҢеҚідҪҝж–Үд»¶дёҚеӯҳеңЁд№ҹдёҚдјҡжҠҘй”ҷпјҲеҝҪз•Ҙжқғйҷҗе’ҢдёҚеӯҳеңЁзҡ„ж–Үд»¶пјүгҖӮ
-r жҲ– -R йҖ’еҪ’еҲ йҷӨзӣ®еҪ•еҸҠе…¶еҶ…е®№пјҲеҢ…жӢ¬еӯҗзӣ®еҪ•пјүгҖӮ
-v жҳҫзӨәеҲ йҷӨзҡ„иҜҰз»ҶдҝЎжҒҜпјҲverboseпјүгҖӮ

иҜҰи§ЈйҖүйЎ№

дәӨдә’ејҸеҲ йҷӨпјҲ-iпјү

  • дҪҝз”Ё -iпјҢеңЁеҲ йҷӨжҜҸдёӘж–Үд»¶жҲ–зӣ®еҪ•д№ӢеүҚдјҡжҸҗзӨәзЎ®и®ӨгҖӮ
  • зӨәдҫӢпјҡ иҫ“еҮәпјҡ

йҖ’еҪ’еҲ йҷӨзӣ®еҪ•пјҲ-r жҲ– -Rпјү**

  • rm й»ҳи®ӨдёҚиғҪзӣҙжҺҘеҲ йҷӨзӣ®еҪ•пјҢеҰӮжһңиҰҒеҲ йҷӨзӣ®еҪ•еҸҠе…¶еҶ…е®№пјҢеҝ…йЎ»дҪҝз”Ё -r жҲ– -RгҖӮ
  • зӨәдҫӢпјҡ
    • еҲ йҷӨ my_folder зӣ®еҪ•еҸҠе…¶жүҖжңүеӯҗж–Үд»¶е’Ңеӯҗзӣ®еҪ•гҖӮ

ејәеҲ¶еҲ йҷӨпјҲ-fпјү

  • дҪҝз”Ё -f еҸҜд»ҘеҝҪз•ҘдёҚеӯҳеңЁзҡ„ж–Үд»¶жҲ–жқғйҷҗйҷҗеҲ¶пјҢзӣҙжҺҘејәеҲ¶еҲ йҷӨгҖӮ
  • зӨәдҫӢпјҡ
    • еҚідҪҝж–Үд»¶дёҚеӯҳеңЁпјҢд№ҹдёҚдјҡжҠҘй”ҷгҖӮ

жҳҫзӨәиҜҰз»ҶдҝЎжҒҜпјҲ-vпјү

  • дҪҝз”Ё -vпјҢеҲ йҷӨж—¶дјҡжҳҫзӨәжӯЈеңЁеҲ йҷӨзҡ„ж–Үд»¶/зӣ®еҪ•гҖӮ
  • зӨәдҫӢпјҡ иҫ“еҮәпјҡ

иӯҰе‘Ҡ

  • дҪҝз”Ё rm е‘Ҫд»Өж—¶йңҖиҰҒж јеӨ–е°ҸеҝғпјҢзү№еҲ«жҳҜз»“еҗҲ -r е’Ң -f йҖүйЎ№пјҢеӣ дёәеҲ йҷӨж“ҚдҪңйҖҡеёёжҳҜдёҚеҸҜйҖҶзҡ„гҖӮ
  • дҫӢеҰӮпјҢд»ҘдёӢе‘Ҫд»ӨдјҡејәеҲ¶еҲ йҷӨж•ҙдёӘзӣ®еҪ•пјҢд»ҘеҸҠе…¶дёӯзҡ„жүҖжңүеҶ…е®№пјҡ еҚұйҷ©жҸҗзӨәпјҡдёҚе»әи®®еңЁжІЎжңүзЎ®и®Өи·Ҝеҫ„зҡ„жғ…еҶөдёӢдҪҝз”Ё rm -rfпјҢе°Өе…¶жҳҜеҜ№зі»з»ҹйҮҚиҰҒзӣ®еҪ•пјҢеҰӮ /гҖӮ

зӨәдҫӢ

еҲ йҷӨеҚ•дёӘж–Үд»¶

  • еҲ йҷӨеҪ“еүҚзӣ®еҪ•дёӢзҡ„ file1.txt ж–Үд»¶гҖӮ

еҲ йҷӨеӨҡдёӘж–Үд»¶

  • дёҖж¬ЎжҖ§еҲ йҷӨеӨҡдёӘж–Үд»¶гҖӮ

еҲ йҷӨзӣ®еҪ•еҸҠе…¶еҶ…е®№

  • еҲ йҷӨ my_folder зӣ®еҪ•еҸҠе…¶жүҖжңүеӯҗж–Үд»¶е’Ңеӯҗзӣ®еҪ•гҖӮ

ејәеҲ¶еҲ йҷӨзӣ®еҪ•еҸҠе…¶еҶ…е®№

  • ејәеҲ¶еҲ йҷӨ my_folder еҸҠе…¶еҶ…е®№пјҢдёҚжҸҗзӨәзЎ®и®ӨгҖӮ

дәӨдә’ејҸеҲ йҷӨ

  • еҲ йҷӨеүҚдјҡиҜўй—®зЎ®и®ӨгҖӮ

еҲ йҷӨеӨҡдёӘж–Үд»¶е№¶жҳҫзӨәиҜҰз»ҶдҝЎжҒҜ

иҫ“еҮәзӨәдҫӢпјҡ

еҲ йҷӨеҪ“еүҚзӣ®еҪ•дёӢжүҖжңүж–Үд»¶

  • еҲ йҷӨеҪ“еүҚзӣ®еҪ•дёӢзҡ„жүҖжңүж–Үд»¶пјҲдёҚеҢ…жӢ¬еӯҗзӣ®еҪ•пјүгҖӮ

еҲ йҷӨеҪ“еүҚзӣ®еҪ•дёӢзҡ„жүҖжңүж–Үд»¶е’Ңзӣ®еҪ•

  • еҲ йҷӨеҪ“еүҚзӣ®еҪ•дёӢзҡ„жүҖжңүж–Үд»¶е’Ңзӣ®еҪ•пјҢеҢ…жӢ¬еӯҗзӣ®еҪ•еҸҠе…¶еҶ…е®№гҖӮ

еёёи§Ғй”ҷиҜҜеҸҠи§ЈеҶіж–№жі•

  1. ж–Үд»¶дёҚеӯҳеңЁ
  • и§ЈеҶіж–№жі•пјҡзЎ®дҝқж–Үд»¶и·Ҝеҫ„жӯЈзЎ®пјҢжҲ–иҖ…дҪҝз”Ё rm -f еҝҪз•ҘдёҚеӯҳеңЁзҡ„ж–Үд»¶гҖӮ
  1. жқғйҷҗдёҚи¶і
  • и§ЈеҶіж–№жі•пјҡдҪҝз”Ё sudo жҸҗжқғеҲ йҷӨпјҡ
  1. иҜҜеҲ йҮҚиҰҒж–Үд»¶
  • еҰӮжһңиҜҜеҲ дәҶж–Үд»¶пјҢйҖҡеёёж— жі•зӣҙжҺҘжҒўеӨҚгҖӮ
  • и§ЈеҶіж–№жі•пјҡ
    • е®ҡжңҹеӨҮд»Ҫж•°жҚ®гҖӮ
    • дҪҝз”Ёж–Үд»¶жҒўеӨҚе·Ҙе…·е°қиҜ•жҒўеӨҚпјҲеҰӮ extundelete жҲ– testdiskпјүгҖӮ

з»“еҗҲе…¶д»–е‘Ҫд»Ө

еҲ йҷӨзү№е®ҡзұ»еһӢзҡ„ж–Үд»¶

  • еҲ йҷӨеҪ“еүҚзӣ®еҪ•дёӢжүҖжңү .txt ж–Үд»¶пјҡ

й…ҚеҗҲ find е‘Ҫд»ӨжҢүжқЎд»¶еҲ йҷӨ

  • еҲ йҷӨеҪ“еүҚзӣ®еҪ•еҸҠеӯҗзӣ®еҪ•дёӯжүҖжңү .log ж–Үд»¶пјҡ

  • еҲ йҷӨ 7 еӨ©еүҚзҡ„ж–Үд»¶пјҡ

жҖ»з»“

  • rm жҳҜдёҖдёӘејәеӨ§дҪҶеҚұйҷ©зҡ„е‘Ҫд»ӨпјҢеҸҜз”ЁдәҺеҲ йҷӨж–Үд»¶е’Ңзӣ®еҪ•пјҢе°Өе…¶еңЁдҪҝз”ЁйҖ’еҪ’е’ҢејәеҲ¶йҖүйЎ№пјҲеҰӮ -r е’Ң -fпјүж—¶йңҖиҰҒзү№еҲ«е°ҸеҝғгҖӮ
  • жҺЁиҚҗдҪҝз”Ё -i зЎ®дҝқеҲ йҷӨеүҚиҝӣиЎҢзЎ®и®ӨпјҢйҒҝе…ҚиҜҜеҲ йҮҚиҰҒж–Үд»¶гҖӮ
  • еҰӮжһңж¶үеҸҠжү№йҮҸеҲ йҷӨж–Үд»¶жҲ–зӣ®еҪ•пјҢе»әи®®е…ҲжЈҖжҹҘи·Ҝеҫ„е’ҢеҶ…е®№пјҢд»Ҙе…ҚйҖ жҲҗдёҚеҸҜжҒўеӨҚзҡ„жҚҹеӨұгҖӮ
